The energy space has been slow to integrate Data and Big Data technologies. The primary obstacle to adoption has been that existing energy data platforms are not accessible, adaptable and definitely not easy to master for basic researchers. Databus was designed to be accessible to a wide range of users from basic researchers to advanced programmers. Databus is easily adapted to evolving data and big data technologies through a set of standard abstractions and protocols. Databus makes the messy task of collecting and analyzing data from energy systems easier.

Databus is Software that enables collecting, storing and requesting time series data. It allows researchers to share that data with their team as well as download manipulated data. There are many modules researchers can use so they can download manipulated data. Some examples include:

  1. Downloading a spline of the data over 60 second interval
  2. Downloading a time average the data
  3. Downloading a clean data set
  4. Simply Downloading the raw data they stored
