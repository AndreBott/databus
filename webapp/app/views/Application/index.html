#{extends 'main.html' /} #{set title:'Home' /} #{set tab:'home' /} If you are new to this system, you have a few options
  1. Have someone add you to a User Group that gives you access to their Schemas and tables
  2. Create a User Group and Schema. Add tables to your Schema and users to your User Group so those Users can access your data
User Groups contain users. Schemas contain tables. You can then assign a User or User Group access to those Schemas or tables. This is useful so that if you want to give Nick all the access that Aaron has, you can just add Nick to the same UserGroup that Aaron is already in and any Schemas or tables that Aaron had access to, Nick now has access to.

User Groups

If you want to be added to a User Group, you need to talk to the User Group's Admin. Group Admins add/remove users to/from a User Group and that is all. #{list groups, as:'group'} #{/list}

Schemas

Schemas contain tables that admins of the schema can give users or groups access to. Some schemas contain 1 to 10 tables while other Schemas contain 30,000 tables. If you want access to one of the schemas, talk to one of the users listed.
